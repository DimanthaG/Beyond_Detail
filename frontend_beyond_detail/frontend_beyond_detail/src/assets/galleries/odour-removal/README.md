# Odour Removal Gallery

Upload your Odour Removal gallery images to this folder.

## How to Use

1. Add your images here (JPG, PNG, WEBP formats)
2. Images will automatically appear on the Odour Removal page gallery
3. No code changes needed - just refresh the page!

## Recommended Settings

- **Format**: JPG or PNG
- **Size**: 1200x800px or larger
- **Naming**: Use descriptive names like $(odour-removal)-1.jpg, $(odour-removal)-result-1.jpg, etc.
