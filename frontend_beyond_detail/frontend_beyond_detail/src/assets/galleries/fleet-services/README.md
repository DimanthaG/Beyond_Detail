# Fleet Services Gallery

Upload your Fleet Services gallery images to this folder.

## How to Use

1. Add your images here (JPG, PNG, WEBP formats)
2. Images will automatically appear on the Fleet Services page gallery
3. No code changes needed - just refresh the page!

## Recommended Settings

- **Format**: JPG or PNG
- **Size**: 1200x800px or larger
- **Naming**: Use descriptive names like $(fleet-services)-1.jpg, $(fleet-services)-result-1.jpg, etc.
