# Headlight Restoration Gallery

Upload your Headlight Restoration gallery images to this folder.

## How to Use

1. Add your images here (JPG, PNG, WEBP formats)
2. Images will automatically appear on the Headlight Restoration page gallery
3. No code changes needed - just refresh the page!

## Recommended Settings

- **Format**: JPG or PNG
- **Size**: 1200x800px or larger
- **Naming**: Use descriptive names like $(headlight-restoration)-1.jpg, $(headlight-restoration)-result-1.jpg, etc.
