# Leather Cleaning Gallery

Upload your Leather Cleaning gallery images to this folder.

## How to Use

1. Add your images here (JPG, PNG, WEBP formats)
2. Images will automatically appear on the Leather Cleaning page gallery
3. No code changes needed - just refresh the page!

## Recommended Settings

- **Format**: JPG or PNG
- **Size**: 1200x800px or larger
- **Naming**: Use descriptive names like $(leather-cleaning)-1.jpg, $(leather-cleaning)-result-1.jpg, etc.
