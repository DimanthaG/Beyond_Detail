# Interior Detailing Gallery

Upload your Interior Detailing gallery images to this folder.

## How to Use

1. Add your images here (JPG, PNG, WEBP formats)
2. Images will automatically appear on the Interior Detailing page gallery
3. No code changes needed - just refresh the page!

## Recommended Settings

- **Format**: JPG or PNG
- **Size**: 1200x800px or larger
- **Naming**: Use descriptive names like $(interior-detailing)-1.jpg, $(interior-detailing)-result-1.jpg, etc.
